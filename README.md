# Arrays-and-Strings

本部分是学习最基本的数据结构**数组** ，因为字符串是由字符数组形成的，所以二者是相似的。
 - 理解二维数组的基本概念和二维数组的使用。
 - 能够运用双指针解决问题 。
   - 同向遍历
   - 向向遍历
